## About

### Online cinema app.

### In addition to watching movies it allows to log in, rate movies, add reviews and edit your personal watch list.

---

## Install

```sh
npm install
```

## Usage

```sh
npm start
```

> - TypeScript
> - React
> - React-Router
> - Redux Toolkit
> - Axios
